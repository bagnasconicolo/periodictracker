Account Creation & Login

  • New Users: Enter a username and password, click "Login & Load" - your account is created automatically
  • Returning Users: Enter your credentials and click "Login & Load"
  • Auto-Save: Use Ctrl+S keyboard shortcut to quickly save changes
  • Session Status: Your login status is shown in the user status bar

üìä The Interactive Periodic Table

Basic Navigation

  • Element Hover: Hover over any element to see its name, atomic number, and brief description
  • Element Selection: Click any element to open its detailed editing panel
  • Visual Status: Elements are color-coded by status:
    • üü¢ Pure - Light green background
    • üü° Representative - Light red background
    • üîµ Alloy - Light blue background
    • üü° Wish - Light yellow background
  • Collection Badges: Elements show small badges for sample count (green) and comment count (red)

Quick Status Mode

  • Enable: Check the "Quick Status" checkbox
  • Select Status: Choose Pure, Representative, Alloy, or Wish from dropdown
  • Apply: Click any element to instantly assign the selected status
  • Bulk Updates: Perfect for quickly categorizing multiple elements

Gallery View

  • Toggle: Check "Gallery View" to display element images as cell backgrounds
  • Visual Preview: See your collection images directly on the periodic table
  • Hover Info: Element details still available on hover

Sticky Table Mode

  • Activate: Click "Stick Table" button to fix the periodic table position
  • Benefits: Table stays visible while scrolling through details
  • Toggle: Click again to return to normal scrolling

üîß Element Details & Editing

Element Information Panel

When you click an element, the right panel shows detailed information and editing options:

  • Status: Set as Pure, Representative, Alloy, or Wish
  • Description: Add personal notes about your sample
  • Quantity: Record amount in grams (supports decimals)
  • Purity: Set purity percentage (0-100%)

Image Management

  • URL Method: Paste image URL and click "Set URL"
  • File Upload: Choose local file (max 2MB) and click "Upload"
  • Preview: Images appear in preview area and are clickable for full-size view
  • Gallery Integration: Images automatically appear in Gallery View mode

üì∑ Sample Management System

Adding Samples

  • Multiple Views: Add various angles, forms, or states of the same element
  • Add Sample: Click "Add Sample" button in element details
  • Sample Details: Each sample can have its own status, description, quantity, and purity
  • Image Options: Use URL or upload file for each sample

Sample Organization

  • Thumbnail View: Sample images display as clickable thumbnails
  • Edit/Delete: Use Edit and Delete buttons for your own samples
  • Status Tracking: Samples can have different statuses than the main element
  • Detailed Records: Track quantity and purity for each individual sample
